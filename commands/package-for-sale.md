# Package Design for Marketplace

Help me package my CNC design for selling online.

1. Ask which marketplace: Etsy, Cults3D, or Gumroad
2. Ask about the design: name, description, material, dimensions
3. Check for existing design files (G-code, SVG, DXF) or offer to generate them
4. Use `package_design` to bundle all files
5. Use `generate_listing_description` to create optimized listing copy
6. If format conversion is needed, use `convert_design_format`

Present the final package contents and listing description for review.
